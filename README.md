# BNB Bank System - Front-End

Bank System Front-end created using Vue.js

### ⚙️ Project setup

Create.env file to setup. (read the env.exemple file)
- 	API URL,
- 	Client ID
- 	Client Token

```
npm install
```

### Run in develop mode

```
npm run serve
```

### Build to production

```
npm run build

```
